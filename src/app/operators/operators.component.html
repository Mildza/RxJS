OPERATORS

RxJS has a whole range of operators that helps you control how the events flow through your observables. Flow control operators are filter, delay, throttleTime, debounceTime, take, takeUntil, distinct, distinctUntilChanged ...

interval

Creates an Observable that emits sequential numbers every specified interval of time

Emits in interval of 1 seconds

Common filtering operators come with interval:

  • take - returns an Observable that emits only the first count values emitted by the source Observable
  • takeLast - emits only the last count values emitted by the source Observable when the source completes.
  • takeUntil - emits the values emitted by the source Observable until a notifier (second) Observable emits a value.
  • takeWhile - emits values emitted by the source Observable so long as each value satisfies the given statement.
