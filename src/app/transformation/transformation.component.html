Transformation operators

map

Applies a given function to each value emitted by the source Observable, and emits the resulting values as an Observable.

Array [1, 2, 3, 4, 5] multpile by 5, we can pass value and second parameter - index

  • mapTo - Emits the given constant value on the output Observable every time the source Observable emits a value.

reduce

Applies an accumulator function over the source Observable, and returns the accumulated result when the source completes, given an optional seed value.

Source observable is of[1, 2, 3, 4] and result is sum of this numebrs.

scan

It's like reduce, but emits the current accumulation whenever the source emits a value.

On every mouse click emit value 1 and accumulate it.

buffer

Collect output values until provided observable emits, emit as array.

Inerval emits every second value, buffer on mouse click catch the emmited values.

  • bufferTime - Collect emitted values until provided time has passed, emit as array.
  • bufferCount - Collect emitted values until provided number is fulfilled, emit as array.
  • bufferWhen - Collect all values until closing selector emits, emit buffered values.
  • bufferToggle - Toggle on to catch emitted values from source, toggle off to emit buffered values as array.
