Combination operators

merge

Turn multiple observables into a single observable.

merge(first, second, third)

concat

Subscribe to observables in order as previous completes, emit values.

sourceOne.pipe(concat(sourceTwo))

combineLatest

When any observable emits a value, emit the latest value from each.

combineLatest(timerOne, timerTwo, timerThree)

race

The observable to emit first is used.

race(timerOne, timerTwo, timerThree)

startWith

Emit given value first.

source.pipe(startWith(0))

Other operators

finalize

Call a function when observable completes or errors.

finalize(() => console.log('Sequence complete'))

repeat

Repeats the stream of items emitted by the source Observable at most count times.

repeat(5)

delayWhen

Delay emitted values determined by provided function.

iif

Decides at subscription time which Observable will actually be subscribed.
